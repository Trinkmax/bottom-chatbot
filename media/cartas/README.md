# Carpeta de Cartas

Esta carpeta debe contener los archivos PDF de las cartas del restaurante.

## Instrucciones

1. Agrega aquí los archivos PDF de las cartas
2. Nombres de archivo requeridos:
   - `carta_antes_00.pdf` - Carta disponible antes de las 00:00
   - `carta_despues_00.pdf` - Carta disponible después de las 00:00

## Estructura requerida:

```
cartas/
├── carta_antes_00.pdf
└── carta_despues_00.pdf
```

El bot enviará automáticamente el PDF correspondiente cuando un cliente solicite ver la carta.
