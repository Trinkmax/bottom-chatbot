# Carpeta de Combos de Cumpleaños

Esta carpeta debe contener las imágenes de los combos de cumpleaños disponibles.

## Instrucciones

1. Agrega aquí las imágenes de los combos (hasta 3 imágenes)
2. Formatos soportados: `.jpg`, `.jpeg`, `.png`, `.gif`
3. Nombres sugeridos:
   - `combo_1.jpg`
   - `combo_2.jpg`
   - `combo_3.jpg`

## Ejemplo de estructura:

```
combos/
├── combo_1.jpg
├── combo_2.jpg
└── combo_3.jpg
```

El bot enviará automáticamente estas imágenes cuando un cliente seleccione la opción "Festejar cumpleaños".
